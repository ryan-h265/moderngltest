WHAT THE FIX DID
───────────────────────────────────────────────────────────────────────────────

OLD CODE:
    is_open, is_selected = imgui.begin_tab_item(category)
    if is_selected:
        draw_grid()  # Only renders if selected

ISSUE:
    Nothing was ever "selected", so draw_grid() never ran

NEW CODE:
    is_open, is_selected = imgui.begin_tab_item(category)
    if tab_idx == 0 and not any_tab_selected and not is_selected:
        is_selected = True  # Force first tab selected
    if is_selected:
        draw_grid()

RESULT:
    Grid always renders on startup, normal tab switching works after
